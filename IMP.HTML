Java Important Questions

Grand Test-1

Loops:

  1. Print the sequence from 1 to 100 using a for loop.
  2. Print the sequence from 100, 98, 96, ... 2, 0 using a for loop.
  3. Print the sequence 1, 2, 4, 8, 16, 32, ... 2048 using a for loop.
  4. Print the alphabet sequence a, b, c, ... x, y, z using a for loop.
  5. Print the sequence a, c, e, ... y using a for loop.
  6. Print the sequence 100, 95, 90, ... 5, 0 using a for loop.

Conditional Statements:

  1. Find the biggest of three numbers using if-else-if.
  2. Check if a character is a vowel using a switch statement.
  3. Determine if a number is negative or positive using an if statement.
  4. Check if a year is a leap year using if-else-if.

Arrays:

  1. Take an integer array of 10 numbers, and print the array in reverse.
  2. Find the sum of array elements using a for loop.
  3. Count how many even and odd numbers are in an array.
  4. Print the sum of even numbers in an array.
  5. Find the second largest number in an array.
  6. Determine how many times each array element is repeating.
  7. Move all 0's to the left side and 1's to the right side in an array containing 0's and 1's.
  8. Copy elements from one array to another using a for loop.
  9. Find common elements between two arrays.
  10. Copy elements of one array into another in reverse order.
  11. Copy elements from two arrays into a third array in the same sequence (first array1, then array2).
  12. Find the difference between the sum of even and odd numbers in an array.
  13. Print the array in reverse order.
  14. Find the biggest element in an array of 10 integers.
  15. Print the sum of numbers present in an array.
  16. Copy even numbers into one array and odd numbers into another from a given array of 10 numbers.
  17. Find the position of a number in an array. Print "number not available" if itâ€™s not present.
  18. Sort an array of 10 numbers in ascending order.

Strings:

  1. Check if a string is a palindrome (do not use the reverse() function).
  2. Replace vowels in a string with the letter 'z'.
  3. Count how many words are in a string.
  4. Print the words of a string in reverse order.

GRAND TEST-2

Even-Odd Based Array Questions

  1. Take an array of 10 numbers. Write logic for printing even numbers present in the array.
  2. Take an array of 10 numbers. Write logic for printing even numbers present at even indexes of the array.
  3. Take an array of 10 numbers. Write logic for printing odd numbers present in the array.
  4. Take an array of 10 numbers. Write logic for printing odd numbers present at odd indexes of the array.
  5. Take an array of 10 numbers. Write logic for calculating the sum of even numbers and the sum of odd numbers present in the array.
  6. Take an array of 10 numbers. Write logic to calculate the average of the numbers present in the array.
  7. Take an array of 10 numbers. Write logic to calculate the average of even numbers present in the array. Write logic to calculate the average of odd numbers present in the array.
  8. Take an array of 10 numbers. Write logic for calculating the difference between the sum of even numbers and the sum of odd numbers present in the array. The difference should be printed as a positive number.
  9. Take an array of 10 numbers. Write logic for printing even numbers and wherever odd numbers are present, you have to print "palle" in place of that number.
  10. Take an array of 10 numbers. Write logic for printing the array numbers which are divisible by both 3 and 5.
  11. Take an array of 10 numbers. Write logic for calculating the sum of numbers which are divisible by both 3 and 5.
  12. Take an array of 10 numbers. Swap adjacent numbers, i.e., the 0th element should be copied into the 1st element and vice versa, the 2nd element should be copied into the 3rd element and vice versa, etc.
  13. Take an array of 10 numbers. Count how many negative numbers are there in the array, and replace those elements with the value 0.
  14. Take an array of 10 numbers. Read a number from the keyboard. Find the position of that number in the array. If the number is not available, you have to print "number is not available." Otherwise, you have to print the position of that number.
  15. Take an array of 10 numbers. Read a number from the keyboard. Find the total occurrences of that number in the array.
  16. Take arr1 of 10 numbers. Take arr2 of 10 numbers. Print common elements of arr1 and arr2.
  17. Take arr1 of 10 numbers. Take arr2 of 10 numbers. Print arr1 elements which are not present in arr2.
  18. Take arr1 of 10 numbers. Take arr2 of 10 numbers. Print arr2 elements which are not present in arr1.
  19. Take arr1 of 10 numbers. Take arr2 of 10 numbers. Print uncommon elements of arr1 and arr2.

Array Copying Based Questions

  1. Take array1 of 10 numbers. Take an empty array2 of the same size as array1. Write logic to copy array1 elements into array2. Don't use Arrays.copy().
  2. Take array1 of 10 numbers. Take an empty array2 of the same size as array1. Write logic to copy every alternate element of array1 into array2.
  3. Take array1 of 10 numbers. Take an empty array2 of the same size as array1. Write logic to copy even indexed elements of array1 into array2.
  4. Take array1 of 10 numbers. Take an empty array2 of the same size as array1. Write logic to copy odd indexed elements of array1 into array2.
  5. Take array1 of 10 numbers. Write logic to shift all elements of array1 to the right by one position. (The last element should be discarded)
  6. Take array1 of 10 numbers. Write logic to shift all elements of array1 to the left by one position. (The first element should be discarded)

Pattern Based Questions

  1. Print a pattern of increasing stars, i.e., 1 star on the first line, 2 stars on the second line, ..., 10 stars on the tenth line.
  2. Print a pattern of decreasing stars, i.e., 10 stars on the first line, 9 stars on the second line, ..., 1 star on the tenth line.
  3. Print a pattern of stars forming a triangle.
  4. Print a pattern of alternating stars and spaces.
  5. Print a pyramid pattern of stars.
